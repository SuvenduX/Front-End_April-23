Block Elements

Block Elements are those which start from the new line . It takes the whole width of the page

Example :

  1. Header
  2. Nav
  3. Aside
  4. Div
  5. Main
  6. Footer

Inline Elements

Inline Elements are those which starts where last element ends . It does not take the whole width of the page

Example :

  • img
  • audio
  • video
  • span
